# GitHub Repository Setup Guide

## 🔗 Add Website Link to Repository

To make your live portfolio visible on the GitHub repository page, follow these steps:

### Step 1: Add Website URL
1. Go to your repository: https://github.com/ShubhamPaithane04/My_Portfolio
2. Click the **⚙️ Settings** button (gear icon) in the top right
3. In the "About" section (right sidebar), click **⚙️ Edit**
4. Add these details:
   - **Website**: `https://my-portfolio-oerl.vercel.app`
   - **Description**: `Interactive macOS-style portfolio built with React - Featuring draggable windows, music player, and mobile responsive design`
   - **Topics**: Add the following tags (press Enter after each):
     - `react`
     - `portfolio`
     - `vite`
     - `framer-motion`
     - `responsive-design`
     - `macos-ui`
     - `javascript`
     - `css3`
     - `frontend`
     - `web-development`
5. Check **✓ Use your GitHub Pages website**
6. Click **Save changes**

### Step 2: Pin Repository (Optional)
1. Go to your profile: https://github.com/ShubhamPaithane04
2. Click **Customize your pins**
3. Select **My_Portfolio** repository
4. Click **Save pins**

### Result
Your repository will now show:
- 🌐 Website link at the top (clickable)
- 📝 Professional description
- 🏷️ Topic tags for discoverability
- ⭐ Featured on your profile (if pinned)

## 📸 Add Screenshots (Optional but Recommended)

Create a folder for screenshots:
```bash
mkdir screenshots
```

Take screenshots of:
1. Desktop view - Full macOS interface
2. Mobile view - Responsive design
3. Individual sections (About, Projects, etc.)

Add to README:
```markdown
## 📸 Screenshots

### Desktop View
![Desktop](screenshots/desktop.png)

### Mobile View
![Mobile](screenshots/mobile.png)
```
